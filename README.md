
# Blockchain 001
It's a simple implementation of Bitcoin in Golang.

## Some features
### Bitcoin with single node
1. UTXO Set
2. Block Mining
3. Merkle Root
4. Coin Transfer 
5. Simple Wallet

### Bitcoin P2P Network
1. Block Synchronization
2. Neighbor Detection
3. Different Nodes: light node, full node and mining node // TO DO: SPV node
